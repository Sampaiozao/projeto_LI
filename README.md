# projeto_LI
Grupo 3


“Gerir as tarefas domésticas e o orçamento da casa tem-se tornado cada vez mais complicado para mim. Preciso de um gestor doméstico que consiga controlar os horários das tarefas, controlar as despesas (água, luz, gás, compras de supermercado, subscrições), lembrar cada membro da família das suas responsabilidades e optimizar as listas de compras com base no que já existe em casa. Deve adaptar-se às nossas rotinas semanais, integrar os calendários partilhados e alertar-nos antes de os pagamentos vencerem, para que a casa funcione de forma organizada sem ser preciso andar sempre a controlar tudo à mão.”

1. Definir o que a app vai fazer (funcionalidades base)
Por exemplo:

Gestão de tarefas:

Criar tarefas domésticas (ex.: lavar loiça, aspirar, levar o lixo).
Atribuir tarefas a membros da casa.
Definir frequência (diária, semanal, mensal).
Mostrar uma lista de tarefas do dia / da semana.
Gestão de despesas básicas:

Registar despesas (água, luz, gás, supermercado).
Categorizar despesas.
Ver total gasto no mês por categoria.
Lembretes:

Notificações ou alertas para tarefas que vencem hoje.
Lembrete para pagamento de contas (data limite).

App web (mais simples para começar):

Frontend: HTML/CSS/JavaScript ou um framework como React
Backend: Node.js, Python, Java (Spring), etc.
Base de dados: SQLite, PostgreSQL, MySQL ou mesmo ficheiros JSON para algo muito simples.
